Sobre mí

Programador web desde hace ya 5 años, desarrollando proyectos web basados en componentes mediante React así como programando aplicaciones web en el servidor mediante PHP y Node.

Amplio conocimiento de frameworks de desarrollo de aplicaciones web como Symfony, Laravel, Express y librerías de front como React.js.

Soy especialista en Wordpress como sistema de gestión de contenidos (CMS) y woocommerce para la creación de tiendas online (E-COMMERCE).

Mis principales clientes son Agencias de Web y Marketing. También trabajo como parte de un equipo de desarrollo en empresas de programación y diseño web.

Me encanta mi trabajo y por ello siempre estoy en constante evolución aprendiendo nuevos lenguajes, buenas prácticas y frameworks.

Fanático de las buenas prácticas y obsesionado con las arquitecturas de software como CQRS, Event Sourcing y Arquitectura Hexagonal. Pienso en la estandarización como algo necesario para organizar el caos en lo que se puede convertir un proyecto web.
